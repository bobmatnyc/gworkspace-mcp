## Quick Tool Finder

### Email Operations
- Search emails: `search_gmail_messages`
- Send email: `send_email`
- Format emails: `format_email_content`, `create_formatted_email`
- See [Gmail API](./gmail.md)

### Calendar Operations
- View events: `get_events`
- Create meeting: `create_event`
- Check availability: `query_free_busy`
- See [Calendar API](./calendar.md)

### File Operations
- Find files: `search_drive_files`
- Upload: `upload_drive_file`
- See [Drive API](./drive.md)

### Document Operations
- Create doc: `create_document`
- Add content: `append_to_document`
- Format text: `format_text_in_document`, `format_paragraph_in_document`
- See [Docs API](./docs.md)

### Spreadsheet Operations
- Create sheets: `create_spreadsheet`
- Format cells: `format_cells`, `set_number_format`
- Create charts: `create_chart`
- See [Sheets API](./sheets.md)

### Presentation Operations
- Create slides: `create_presentation`
- Format text: `format_text_in_slide`
- Add content: `add_formatted_text_box`, `add_image`
- See [Slides API](./slides.md)

### Task Operations
- List tasks: `list_tasks`
- Create task: `create_task`
- See [Tasks API](./tasks.md)
